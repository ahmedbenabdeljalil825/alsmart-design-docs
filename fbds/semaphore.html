Semaphore

Compressor Control Advanced FunctionBlock v1.2.3

FBD Diagram

semaphore.png

Description



The Semaphore block manages the turning on and off of compressors until 4 circuits, to respect the minimum ON (uiMinOnInterval) and OFF (uiMinOffInterval) time, in particular:
1) if xReqOnCx is TRUE the output xEnOnCx becomes TRUE if uiMinOnInterval time is elapsed; 2) if xReqOffCx is TRUE the output xEnOffCx becomes TRUE if uiMinOffInterval time is elapsed; 3) the first xReqOnCx = TRUE is served until its value becomes FALSE, then the next request xReqOnCy will be examined, where y = x+1, 1 if goes above 4;
4) same behaviour for xReqOffCx, when TRUE it is served until its value becomes FALSE.
The increasing of usiStagesOn produces a restarting of timer to count uiMinOnInterval.
If xEnable = FALSE then all outputs becomes TRUE.
