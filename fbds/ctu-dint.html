CTU_DINT

Inputs & Sensors Intermediate FunctionBlock v1.0.0

FBD Diagram

ctuscheme.png

ctu48.png (Icon)

Description

(Same as CTU but with PV and CV of DINT data type)

The CTU_DINT block counts up for each FALSE-to-TRUE transition of the input CU and sets the output Q when the accumulated value (CV) reaches the preset value (PV) value.

  • rising edge of R reset CV to 0
  • rising edge of CU increments CV by 1
  • when CV reaches PV the output Q is set
  • once CV is equal to PV its value remains until a rising edge of R

Inputs (3)

Name Type Init Description
CU BOOL Count-up on input rising edge
R BOOL Reset counter input
PV DINT Preset value

Outputs (2)

Name Type Description
Q BOOL Count-up output
CV DINT Counter current value

Local Variables (2)

Name Type Description
CUEDGE BOOL
MEMCU BOOL

Source Code (IL)

(*	Check for reset input *)
	LD		R
	JMPCN	l0
	
	LD		0
	ST		CV
	LD		FALSE
	ST		MEMCU
	JMP		l1

	(*	Check for increment enable *)
l0:
	LD		CU
	ANDN 	MEMCU
	ST 		CUEDGE
	
	LD 		CU
	ST 		MEMCU
	
	LD 		CUEDGE
	AND(	CV
	LT		16#7FFFFFFF
	)
	JMPCN	l1
	
	LD		CV
	ADD		1
	ST		CV
	
l1:

	(*	Counter output	*)
	LD		CV
	GE		PV
	ST		Q
