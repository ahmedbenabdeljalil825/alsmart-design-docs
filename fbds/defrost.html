Defrost

Defrost Logic Advanced FunctionBlock v1.2.3

FBD Diagram

defrost.jpg

Description



This block manages the Defrost phase; in particular if input value rInputProbe is below rStartSetpoint for a time of udiStartVerifyingTime the output xDefrost becomes TRUE;
to exit from Defrost phase, the input value rInputProbe must go above rStopSetpoint, with timeout udiDefrostMaximumTime.
If the Defrost ends for timeout and the input bDefrostType = 0 then the output xEndForTimeOutAlarm is set; if bDefrostType = 1 means that the end for timeout
is the rule, so the xEndForTimeOutAlarm is not set.

Then, the successive Defrost phase will happen after udiMinTimeBetweenDefrostSession with exception if the rInputProbe goes below rForcedDefrostStartSetpoint
for udiForcedDefrostVerifyingTime time; in this case the Defrost phase is forced active by setting xDefrost = TRUE.
The force of defrost is enabled only if udiForcedDefrostVerifyingTime is greater than 0 seconds.
