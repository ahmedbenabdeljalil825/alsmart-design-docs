Check_AI_Err

Alarms & Safety Intermediate FunctionBlock v1.0.1

FBD Diagram

imgalarm.png (Icon)

Description

When the sensor value is AI_VALUE_IN_ERROR, it means that the sensor is in error. This function block returns the last valid value for the sensor.

Inputs (3)

Name Type Init Description
xEnable BOOL BOOL, range[FALSE,TRUE]. Enable the function
rInputValue REAL REAL. Input sensor value
xIsInAlarm BOOL BOOL, range[FALSE,TRUE]. TRUE: return the last valid sensor value

Outputs (2)

Name Type Description
rSensorValue REAL REAL. Sensor value
xSensorFault BOOL BOOL, range[FALSE,TRUE]. TRUE: sensor fault

Source Code (ST)

IF NOT xEnable THEN

    rSensorValue := rInputValue;
	xSensorFault := FALSE;
	RETURN;

END_IF;


IF xIsInAlarm THEN
	(*	Alarm ON: set the current value	*)
	rSensorValue := rInputValue;
ELSE

	(*	If the analog input is in error: return the last valid value	*)
	IF rInputValue <> AI_VALUE_IN_ERROR THEN
		rSensorValue := rInputValue;
	END_IF;

END_IF;


(*	Check the alarm condition	*)
IF rInputValue = AI_VALUE_IN_ERROR THEN            
	xSensorFault := TRUE;	
ELSE    
    xSensorFault := FALSE;    
END_IF;
