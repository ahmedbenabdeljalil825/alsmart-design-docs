InversionValve

Compressor Control Intermediate FunctionBlock v1.2.3

FBD Diagram

imginversionvalve.png (Icon)

inversionvalve.png

Description



If xModeRequest is FALSE means that Chiller mode (Cooling) is requested, if TRUE the HeatPump mode (Heating) is requested.
The xDefrostRequest = TRUE works only if xModeRequest is FALSE (Cooling).

When xModeRequest becomes TRUE (HeatPump or Heating) from FALSE (Chiller or Cooling) or viceversa:
1) if xCompressorStatus = TRUE (it means that one or more compressors are active) then xActiveMode maintains its status, while xStopCompressor becomes TRUE (to require the turning off of compressors),
when xCompressorStatus becomes FALSE then xActiveMode follows xModeRequest after a delay of uiInversionDelay seconds, after another additional delay of uiInversionDelay the xStopCompressor becomes FALSE.
2) if xCompressorStatus = FALSE (it means that the compressors are in Off) then immediately xStopCompressor becomes TRUE, while xActiveMode follows xModeRequest after a delay of uiInversionDelay seconds, after another additional delay of uiInversionDelay the xStopCompressor becomes FALSE.

If xDefrostRequest becomes TRUE with xModeRequest TRUE (defrost request works only in Heat Pump), the output xActiveDefrost becomes TRUE immediately, then:
1) if xCompressorStatus = TRUE (it means that one or more compressors are active) then xActiveMode does not change its status, while xStopCompressor becomes TRUE,
when xCompressorStatus becomes FALSE then xActiveMode becomes FALSE after a delay of uiInversionDelay, after another additional delay of uiInversionDelay the xStopCompressor becomes FALSE.

2) If xCompressorStaus is FALSE the sequence is as in previous point without waiting for the compressor to stop.

Inputs (4)

Name Type Init Description
xModeRequest BOOL BOOLEAN, range [FALSE,TRUE]. If FALSE means Cooling, if TRUE Heating
xDefrostRequest BOOL BOOLEAN, range [FALSE,TRUE]
xCompressorStatus BOOL BOOLEAN, range [FALSE,TRUE]
uiInversionDelay UINT UNSIGNED INTEGER, unit [s], range [0,30000]. It is the delay from stop of compressor before commutating the valve

Outputs (3)

Name Type Description
xActiveMode BOOL BOOLEAN, range [FALSE,TRUE]. If FALSE it means Cooling, if TRUE Heating
xActiveDefrost BOOL BOOLEAN, range [FALSE,TRUE]. If TRUE it means defrost in progress
xStopCompressor BOOL BOOLEAN, range [FALSE,TRUE]. If TRUE it is required to stop the compressor/s
