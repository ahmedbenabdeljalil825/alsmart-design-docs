PreventionControl

Alarms & Safety Intermediate FunctionBlock v1.2.3

FBD Diagram

preventioncontrol.png

Description



If xEnable is FALSE or xSensorError is TRUE, then the xPreventionStatus is FALSE.

If xEnable is TRUE and xSensorError is FALSE, the function is active.

If xDirection is FALSE then the xPreventionStatus is TRUE if rSensorValue is equal or above rSetpoint, the xPreventionStatus comes back to FALSE if rSensorValue goes below rSetpoint - rDifferential .

if xDirection is TRUE then the xPreventionStatus is TRUE if rSensorValue is equal or below rSetpoint, the xPreventionStatus comes back to FALSE if rSensorValue goes above rSetpoint + rDifferential .

Inputs (6)

Name Type Init Description
xEnable BOOL BOOL, range [FALSE,TRUE]. It enables the function
xDirection BOOL BOOL, range [FALSE,TRUE]. It defines from which direction the SensorValue reaches the Setpoint to activate PreventionStatus
rSensorValue REAL REAL, unit [°C, or BAR, or °F...], range [-3.4E38,3.4E38]
xSensorError BOOL BOOL, range [FALSE,TRUE]. If TRUE the function is disabled
rSetpoint REAL REAL, unit [°C, or BAR, or °F...], range [-3.4E38,3.4E38]
rDifferential REAL REAL, unit [°C, or BAR, or °F...], range [-3.4E38,3.4E38]

Outputs (1)

Name Type Description
xPreventionStatus BOOL BOOL, range [FALSE,TRUE]. If TRUE means the xPreventionStatus is active
