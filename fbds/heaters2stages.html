Heaters2Stages

HMI & Display Intermediate FunctionBlock v1.2.3

FBD Diagram

heaters2stages.png

Description



If xEnable = FALSE then the outputs xOut1 and xOut2 are both FALSE.

If xEnable = TRUE then the outputs depend from xBinaryMode and usiNumOfStages, as follow.

With xBinaryMode = FALSE:
- if usiNumOfStages = 1 then xOut2 is always FALSE, while xOut1 is TRUE if rPowerRequest is 100.0, becomes FALSE when rPowerRequest is 0.
- if usiNumeOfStages = 2 then xOut1 is TRUE if rPowerRequest is equal or greater than 50.0%, xOut2 is TRUE if rPowerRequest =100.0%,
then xOut2 becomes FALSE at 50%, and xOut1 at 0%.

With xBinaryMode = TRUE, the usiNumeOfStages is not considered and both outputs always work, in particular:
- xOut1 is TRUE if rPowerRequest is equal or greater than rThreshold1 but lower that rThreshold2; at rThreshold2 xOut1 becomes FALSE and if xPowerRequest reaches 100% it becomes again TRUE.
With reduction of rPowerRequest, xOut1 becomes FALSE below rThreshold2,and it becomes again TRUE below rThreshold1, finally FALSE at 0%.

- xOut2 is TRUE if rPowerRequest is equal or greater than rThreshold2, then becomes FALSE if rPowerRequest is lower than rThreshold1.
