SMVInitConfiguration

Valve Control Advanced Function v1.0.0

Description

With this function it is possible to change some default configuration values of the valve. This function must be called in the Init task

Return Value

BOOL

Inputs (11)

Name Type Init Description
device Device_t Device_t. Device ID
peripheral Device_t Peripheral_t. Peripheral ID
xEnable BOOL TRUE BOOL, range [FALSE, TRUE]. TRUE: enable the valve
uiNeutralZone UINT 0 UINT, range [0.0,10.0], unit [0.1%] Neutral zone percentage
usiValveODOnPowerFailurePerc USINT 0 USINT, range [0,100], unit [%]. Valve open degree percentage on power failure
usiEmergencyClosingSpeedPerc USINT 100 USINT, range [50,200], unit [%]. Emergency closing speed in percentage
usiCurrentBoostOdStopPerc USINT 100 USINT, range[0,100], unit [%]. Current boost OD stop in percentage
usiCurrentBoostPerc USINT 100 USINT, range[100,150], unit [%]. Current boost percentage
usiCurrentBoostMaxTime USINT 0 USINT, unit [s]. Current boost max time
xOpenWireFaultEnable BOOL FALSE BOOL, range [FALSE, TRUE]. FALSE: Off TRUE: On
usiThermalProtectionPerc USINT 100 USINT, range[0,100], unit [%]. Thermal protection percentage (100 = disabled)

Outputs (0)

Name Type Description
No outputs.

Local Variables (1)

Name Type Description
usiIndex USINT

Source Code (ST)

SMVInitConfiguration := FALSE;

FOR usiIndex := 0 TO CONST_IOSMVLastIndex DO

	IF ((SMVDriver[usiIndex].device = device) AND
		(SMVDriver[usiIndex].peripheral = peripheral)) THEN

		(*	SMV present	*)
		SMVDriver[usiIndex].xEnabling := xEnable;
		SMVDriver[usiIndex].uiNeutralZone := uiNeutralZone;
		SMVDriver[usiIndex].usiValveODOnPowerFailurePerc := usiValveODOnPowerFailurePerc;
		SMVDriver[usiIndex].usiEmergencyClosingSpeedPerc := usiEmergencyClosingSpeedPerc;
		SMVDriver[usiIndex].usiCurrentBoostPerc := usiCurrentBoostPerc;
		SMVDriver[usiIndex].usiCurrentBoostOdStopPerc := usiCurrentBoostOdStopPerc;
		SMVDriver[usiIndex].usiCurrentBoostMaxTime := usiCurrentBoostMaxTime;
		SMVDriver[usiIndex].xOpenWireFaultEnable := xOpenWireFaultEnable;
		SMVDriver[usiIndex].usiThermalProtectionPerc := usiThermalProtectionPerc;

		SMVInitConfiguration := TRUE;
		
		RETURN;
	END_IF;
END_FOR;
