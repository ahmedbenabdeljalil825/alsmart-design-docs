CTUD_DINT

Inputs & Sensors Intermediate FunctionBlock v1.0.0

FBD Diagram

ctudscheme.png

ctud48.png (Icon)

Description

(Same as CTUD but with PV and CV of DINT data type)

The CTUD_DINT block counts up for each FALSE-to-TRUE transition of the input CU and count down for each FALSE-to-TRUE transition of the input CD.
Sets the QU output when the accumulated value (CV) reaches the preset value (PV) value and sets the QD output when the accumulated value (CV) becomes 0.

  • rising edge of R reset CV to 0
  • rising edge of LD set CV to PV
  • rising edge of CU increments CV by 1
  • rising edge of CD decrements CV by 1
  • when CV reaches PV the output QU is set
  • when CV becomes 0 the output QD is set

Inputs (5)

Name Type Init Description
CU BOOL Count-up input
CD BOOL Count-down input
R BOOL Reset counter input
LD BOOL Preset counter input
PV DINT Preset value

Outputs (3)

Name Type Description
QU BOOL Count-up output
QD BOOL Count-down output
CV DINT Counter current value

Local Variables (4)

Name Type Description
MEMCU BOOL Previous value of CU
MEMCD BOOL Previous value of CD
CUEDGE BOOL True whenever a posedge on CU is detected
CDEDGE BOOL True whenever a posedge on CD is detected

Source Code (IL)

(* Check for reset input *)
	LD		R
	JMPCN	l0
	
	LD		0
	ST		CV
	LD		FALSE
	ST		MEMCU
	LD		FALSE
	ST		MEMCD
	JMP		l10

l0:
	(* Check for preset input *)
	LD		LD
	JMPCN	l1
	
	LD		PV
	ST		CV
	JMP		l10

l1:
	(* Check for posedge on CU *)
	LD		CU
	ANDN 	MEMCU
	ST 		CUEDGE
	(* Store CU value for next cycle *)	
	LD 		CU
	ST 		MEMCU

	(* Check for posedge on CD *)
	LD		CD
	ANDN 	MEMCD
	ST 		CDEDGE
	(* Store CD value for next cycle *)	
	LD 		CD
	ST 		MEMCD

	(* Check for simultaneous posedges on *)
	LD		CUEDGE
	AND		CDEDGE
	JMPC	l10
	
	(* ASSERT(!CUEDGE || !CDEDGE) *)

	(* Check if posedge on CU and we are in bounds *)
	LD		CUEDGE
	AND(	CV
	LT		16#7FFFFFFF
	)
	JMPCN	l2
	
	(* ASSERT(CUEDGE) *)
	(* Increment *)
	LD		CV
	ADD		1
	ST		CV

l2:
    (* Check if posedge on CD and we are in bounds *)
	LD		CDEDGE
	AND(	CV
	GT		-2147483648
	)
	JMPCN	l10
	
	(* ASSERT(CDEDGE) *)
	(* Decrement *)
	LD		CV
	SUB		1
	ST		CV
	
	(* ASSERT(!CUEDGE && !CDEDGE) *)
	
l10:
	(* Up counter output *)
	LD		CV
	GE		PV
	ST		QU

	(* Down counter output *)
	LD		CV
	LE		0
	ST		QD
