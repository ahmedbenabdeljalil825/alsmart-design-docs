LiqValveManager

Compressor Control Advanced FunctionBlock v1.2.3

FBD Diagram

liqvalvemanager.png

Description



The main function of LiqValveManager is to turn on and off the compressor with a delay time compared to switching on and off of the liquid valve.

If xEnable = FALSE, the output xValveEn = FALSE and usiOnOffEn = 1 to allow the compressor to start.
If xEnable = TRUE the the LiqValveManager manages one output for valve (xValveEn) and one output for compressor (usiOnOffEn).

The xValveEn becomes TRUE without delay when usiReqOnOff = 1, it becomes FALSE when usiReqOnOff = 0 or 2.

The usiOnOffEn is 1 with time delay compared to the input usiReqOnOff = 1, this delay can be at max equal to input uiStartUpMaxTime, but it can be less or 0
if the input for pressure rSuctPress is enabled by xEnableSuctPress=TRUE, it is not in alarm by xSuctPressAlarm=FALSE and the pressure value rSuctPress is lower than starting reference set rStartUpPressSet. The usiOnOffEn goes to 2 with time delay compared to the input usiReqOnOff = 2, this delay can be at max equal to input uiPumpDown, but it can be less or 0
if the input for pressure rSuctPress is enabled by xEnableSuctPress=TRUE, it is not in alarm by xSuctPressAlarm=FALSE and the pressure value (rSuctPress) is greater than reference set rPumpDownPressSet.

Name Type Init Description

usiReqOnOff USINT USINT, range [0,2]. If 1 then is requested to Liquid valve to start, if 2 is requested to stop.
uiPumpDownMaxTime UINT USINT, unit [s], range [0,65535]
rPumpDownPressSet REAL REAL, unit [BAR or PSI], range [-1.0, 3.4E38]
uiStartUpMaxTime UINT UINT, unit [s], range [0,65535]
rStartUpPressSet REAL REAL, unit [BAR or PSI], range [-1.0, 3.4E38]
xEnableSuctPress BOOL BOOL, range [FALSE,TRUE]. If TRUE the pressure is evaluated.
rSuctPress REAL REAL, unit [BAR or PSI], range [-1.0, 3.4E38]
xSuctPressAlarm BOOL BOOL, range [FALSE,TRUE]. If TRUE means the pressure is not valid

Outputs (2)

Name Type Description
xValveEn BOOL BOOL, range [FALSE,TRUE].
usiOnOffEn USINT USINT, range [0,2]. 0 to avoid the stop of compressor waiting for pump down of compressor, 1 to allow the start, 2 to allow the stop
