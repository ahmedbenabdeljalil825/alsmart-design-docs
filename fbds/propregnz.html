PropRegNZ

Control Logic Intermediate FunctionBlock v1.2.3

FBD Diagram

proportionalregulation.png

prop_reg_nz.png

Description





If DIRECTION = FALSE and the IN is temperature then the mode is COOL, OUT increases when VALUE increases above SETPOINT,
with DIRECTION = TRUE the mode is HEAT, OUT increases when VALUE decreases below SETPOINT.

This regulation can be applied with inputs different from temperature as pressure, or other.
Picture shows the COOLING mode with FORCE_ZERO_IN_NZ respectively FALSE and TRUE.

With NEUTRAL_ZONE equal to 0 then the OUT goes from 0 at SETPOINT to 1000 at SETPOINT + DIFFERENTIAL linearly,
while with a NEUTRAL_ZONE different from 0, there is an hysteresis at restart with minimum OUT value depending from the NEUTRAL_ZONE value.

If FORCE_ZERO_IN_NZ = TRUE the hysteresis is canceled and OUT moves linearly between SETPOINT+NEUTRAL_ZONE and SETPOINT+DIFFERENTIAL.

With DIRECTION = TRUE (HEATING) the diagram is opposite.
