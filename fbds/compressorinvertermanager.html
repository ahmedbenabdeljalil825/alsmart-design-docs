CompressorInverterManager

Compressor Control Advanced FunctionBlock v1.2.3

FBD Diagram

compressorinvertermanager.jpg

Description



This block manages mainly the Inverter Compressor, considering the starting speed (rMinimumStartSpeed) and the respect of minimum time Off (uiMinTimeOff) and minimum time On (uiMinTimeOn).
The inverter compressor can be enabled to start only if Liquid valve is active; this is settable from input usiEnOnOff,
also the Sequencer can delay the start by input usiReqSequenceOnOff.
Then, in case of additional digital compressor it divides the power request (rPowerRequest) in two parts; the first for Inverter Compressor, the second part for the other digital comporessors. It enables the eventual Liquid Valve with output usiReqOnOff , and the eventual sequencer with output usiEnSequencerOnOff.
