Simple_PI_Reg

Control Logic Intermediate FunctionBlock v1.0.0

FBD Diagram

simple_pi_reg.png

Description



If xEnable is FALSE the rOut is 0.

If xEnable is TRUE, rOut is the sum of Proportional, Integral (if enabled) and Derivative (if enabled) components.
If rOut > 0 then rIn is below rSetpoint (if rIn is a temperature it means heating request), while if rOut < 0 then rIn is above rSetpoint (cooling request).

Inputs (7)

Name Type Init Description
xEnable BOOL BOOLEAN, range [FALSE,TRUE]. If TRUE enables the function, if FALSE the OUT = 0
xDirection BOOL BOOLEAN, range [FALSE,TRUE]. If FALSE (COOL) then rOut increases with increment of rValue above rSetpoint, if TRUE (HEAT) then rOut increases with decrement of rValue below rSetpoint
xRegType BOOL BOOLEAN, range [FALSE,TRUE]. If FALSE the regulation is Proportional, if TRUE is Proportionl + Integral
rSetpoint REAL REAL, unit [°C, or °F, or rH%, or BAR, or PSI...], range [-3.4E38,3.4E38]
rDifferential REAL REAL, unit [°C, or °F, or rH%, or BAR, or PSI...], range [-3.4E38,3.4E38]
rValue REAL REAL, unit [°C, or °F, or rH%, or BAR, or PSI...], range [-3.4E38,3.4E38]
uIntegralTime UINT UNSIGNED INTEGER, unit [s], range[0,65535]

Outputs (1)

Name Type Description
rOut REAL REAL, unit [°C, or °F, or rH%, or BAR, or PSI...], range [-3.4E38,3.4E38]

Name Type Description
