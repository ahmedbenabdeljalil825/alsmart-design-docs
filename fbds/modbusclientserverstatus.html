ModbusClientServerStatus

Communication (Modbus) Intermediate FunctionBlock v1.1.0

FBD Diagram

modbusclientserverstatus.png

Description



This function block must be used for each Server device of the Modbus Client network and allows you to know the status of communication between the Client and the Server.
If there is no communication between Client and the Server, after a settable number of attempts (reads/writes), that can be set in Input uiServerNoResponseMaxCount (for example 10), the driver stops readings/writes with the Server so as not to unnecessarily overload the network, the Function Block sets the xMBCommunicationFault value to True.
After a settable number of seconds (Input uiPeriodToRetryToRead, for examle = 60 sec) the Driver tries to reconnect to the Server, making a maximum of uiServerNoResponseMaxCount attempts. If communication resumes, Output xMBCommunicationFault returns to FALSE and reads/writes can continue as normal.

About the Outputs enmMBStatus and enmMBerror, you can find the structure of these Enums in Library Tree, in the AIModbusRTU library, Definitions/Enumeratives node.

Source Code (ST)

(*
	Check the limits for Inputs
*)

IF uiServerNoResponseMaxCount < 5 THEN
	uiServerNoResponseMaxCount := 5;
ELSIF uiServerNoResponseMaxCount > 100 THEN
	uiServerNoResponseMaxCount := 100;
END_IF;

IF uiPeriodToRetryToRead < 10 THEN
	uiPeriodToRetryToRead := 10;
END_IF;


(*
	Logic
*)

xMBCommunicationFault := FALSE;


IF sysMbClientRtuNetElement.status = MbStatus#MbStatus_NotConfigured THEN

	xMBCommunicationFault := TRUE;

ELSE

	IF sysMbClientRtuNetElement.pres = FALSE AND sysMbClientRtuNetElement.slaveNoResponseCount >= uiServerNoResponseMaxCount THEN
		
		xMBCommunicationFault := TRUE;	

	END_IF;

END_IF;


IF xMBCommunicationFault = TRUE THEN 

	IF xMBCommunicationFault <> xLastMBCommFault THEN
		xRetVal := timerFromServerLoss.Start();
		
		usiServerHandle := sysMbMRtu_GetSlaveHandle(sysMbClientRtuNetElement.chn, sysMbClientRtuNetElement.adr);
		xRetServerAct := sysMbMRtu_SlaveActivation(sysMbClientRtuNetElement.chn, usiServerHandle, FALSE);
		
	END_IF;
	
	IF timerFromServerLoss.ElapsedSec() >= uiPeriodToRetryToRead THEN

		usiServerHandle := sysMbMRtu_GetSlaveHandle(sysMbClientRtuNetElement.chn, sysMbClientRtuNetElement.adr);
		xRetServerAct := sysMbMRtu_SlaveActivation(sysMbClientRtuNetElement.chn, usiServerHandle, TRUE);
	
	END_IF;

END_IF;

enmLastMBError := sysMbClientRtuNetElement.error;

enmMBStatus := sysMbClientRtuNetElement.status;
enmMBError := sysMbClientRtuNetElement.error;

xLastMBCommFault := xMBCommunicationFault;
