sysMbMRtu_FC01

Communication (Modbus) Advanced FunctionBlock v1.0.0

Description

Read Coil Status - Function 01 (0x01). This function block can be used in the Background task only. You have to assure that all input variables are properly set before invoking the function block. In particular, the addr input variable must be set to the physical address of an existing Modbus Slave target of this operation. After function block execution, unless an error occurred, you can use a subset of its output variables (from object1 to objectN, where N equals the value of no input variable) to get the Modbus objects read from the communication bus. If an error did occur, you can read its code from the errorCode output variable.
