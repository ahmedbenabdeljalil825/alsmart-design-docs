Pid

Control Loops (PID) Advanced FunctionBlock v1.2.3

FBD Diagram

pid.jpg

Description



If xEnable is FALSE rOut is 0.

If xEnable is TRUE, rOut is the sum of Proportional, Integral (if enabled) and Derivative (if enabled) components.
If rOut > 0 then rIn is below rSetpoint (if rIn is a temperature it means heating request), while if rOut < 0 then rIn above rSetpoint (cooling request).
