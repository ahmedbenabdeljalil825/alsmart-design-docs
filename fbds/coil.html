Coil

Compressor Control Advanced FunctionBlock v1.2.3

FBD Diagram

coil.jpg

Description



This block manages the external coil.

If xEnable = FALSE the outputs xOutputEnable = FALSE and rFixedPower = 0%.

If xEnable = TRUE there are different cases depending from inputs, in particular with xProbeError = FALSE:
- bRegulationMode = 0 produces the outputs always in OFF, xOutputEnable = FALSE, rFixedPower = 0.0%.
- bRegulationMode = 1 produces the outputs always in ON, xOutputEnable = TRUE, rFixedPower = 100.0%.
- bRegulationMode = 2 produces the outputs always in ON with xCompressorStatus = TRUE.
- bRegulationMode = 3 produces the outputs xOutputEnable = TRUE and rFixedPower = 0.0%.
- bRegulationMode = 4 produces the outputs xOutputEnable = TRUE and rFixedPower equal to rFixedLowSpeed or rFixedHighSpeed depending from rExternalTemperature,
if this temperature is above rExtTempThreshold + rExtTempHystereis then rFixedPower = rExtHighSpeed, when it goes below rExtTempThreshold then rFixedPower = rExtLowSpeed.
In all cases if the xOffWithCompressorsOff = TRUE the outputs are OFF in xCompressorStatus = FALSE.

If xEnable = TRUE and xProbeError = TRUE (the probe in error is not relative to rExternalTemperature)
only for bRegulation = 3, then the outputs depend from bBehaviorWithProbeFault value, in particular:
- bBehaviorWithProbeFault = 0 produces all outputs in OFF, xOutputEnable = FALSE, rFixedPower = 0.0. - bBehaviorWithProbeFault = 1 produces all outputs in ON with xCompressorStatus = TRUE, xOutputEnable = TRUE, rFixedPower = 100.0. - bBehaviorWithProbeFault = 2 produces all outputs in ON, xOutputEnable = TRUE, rFixedPower = 100.0.

Inputs (12)

Name Type Init Description
xEnable BOOL BOOL, range [FALSE,TRUE].
usiRegulationMode USINT USINT, range [0,4]. If 0 the outputs are always OFF, if 1 always ON, if 2 are ON with if xCompressorStatus = TRUE, if 3 and xProbeError = FALSE then xOutEnable = TRUE and 0%, if 4 are TRUE at fixed %.
xCompressorStatus BOOL BOOL, range [FALSE,TRUE].
xOffWithCompressorOff BOOL BOOL, range [FALSE,TRUE]. In all bRegulationMode if it is TRUE the output could be active only if xCompressorStatus = TRUE
uiDelayFromCompressorsOff UINT UINT, unit [s], range [0,30000]. NOT USED
xProbeError BOOL BOOL, range [FALSE,TRUE].
usiBehaviorWithProbeFault USINT BYTE, range [0,2]. It works only if xProbeError = TRUE and bRegulationMode = 3. If 0 the outputs are OFF, if 1 are ON if xCompressorStatus ON, if 2 all ON
rFixedLowSpeed REAL REAL, unit [%], range [0, 100.0]
rFixedHighSpeed REAL REAL, unit [%], range [0, 100.0]
rExternalTemperature REAL REAL, unit [°C, or °F], range [-3.4E38,3.4E38]
rExtTempThreshold REAL REAL, unit [°C, or °F], range [-3.4E38,3.4E38]
rExtTempHysteresis REAL REAL, unit [°C, or °F], range [-3.4E38,3.4E38]
