Inverter

Description



If xEnable is FALSE or xAlarm is TRUE then the xStatus is FALSE and rOutPow is 0.

If xEnable is TRUE and xAlarm is FALSE, then the rOutPow normally follows the rPowReq, xStatus is TRUE if rOutPow is greater than 0.

The rOutPower has some exceptions where it is different from rPowReq, in particular:
1) it is limited between rMinSpeed and rMaxSpeed, if rPowReq is greater than 0 but below rMinSpeed then rOutPow goes to rMinSpeed, similar for rMaxSpeed;
2) if rOutPow is equal to 0, the first starting requiires rPowReq to be above rStartSpeed, in this case rOutPow goes to rMaxSpeed for uiStartTime seconds, then it goes to rPowReq;



3) every uiPeriodMaxSpeed minutes the output rOutPow reaches the rMaxSpeed for uiDurationMaxSpeed seconds, then it comes back to rPowReq.

Inputs (9)

Name Type Init Description
xEnable BOOL BOOLEAN, range [FALSE,TRUE]. If TRUE the function is enabled, if FALSE then xStatus = FALSE, uiOutPow = 0
xAlarm BOOL BOOLEAN, range [FALSE,TRUE]. If TRUE then xStatus = FALSE, uiOutPow = 0
rMinSpeed REAL REAL, unit [%], range [0.0,100.0]
rMaxSpeed REAL REAL, unit [%], range [0.0,100.0]
uiStartTime UINT UNSIGNED INT, unit [s], range [0,30000]
rStartSpeed REAL REAL, unit [%], range [rMinSpeed, rMaxSpeed]
uiPeriodMaxSpeed UINT UNSIGNED INT, unit [min], range [0,30000]
uiDurationMaxSpeed UINT UNSIGNED INT, unit [s], range [0,30000]
rPowReq REAL REAL, unit [%], range [0.0,100.0]

Outputs (2)

Name Type Description
xStatus BOOL BOOLEAN, range [FALSE,TRUE]. It is TRUE when rOutPow greater than 0
rOutPow REAL REAL, unit [%], range [0.0,100.0]
